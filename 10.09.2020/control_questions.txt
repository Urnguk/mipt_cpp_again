С++ - компилируемый, то есть он сначала переводит исполняемый код в машинный, что дает преимущество по скорости исполнения программы (и вроде отладки, так как при компиляции будут сразу вычислены некоторые ошибки). Помимо этого он позволяет использовать разные парадигмы программирования (функциональное, Объектно-ориентированное и т.д.). В нем также присутствует статическая типизация, которая жестко закрепляет за функциями, переменными и прочими объектами тип данных, который в них содержится (ими возвращается), причем он никак не может быть изменен после объявления.
Фундаментальные типы данных: bool - логический тип данных, может принимать значения 0 и 1 (true | false). Обычно кодируется одним байтом, хотя используется только один бит; char - используется для хранения отдельно взятого символа. Обычно кодируется одним байтом; int - используется для хранения целых чисел. Обычно кодируется четырьмя байтами, хотя есть вариации, учитывающие и не учитывающие знак числа, а также с большим или меньшим количество выделенной памяти; float - используется для хранения дробных чисел(с плавающей точкой). Аналогично int, обычно кодируется четырьмя байтами, но обладает вариациями; void - пустота. Тип данных, который ничего не возвращает(например можно использовать для функций, являющихся процедурами). 
Проблема переносимости возникает при использовании кода, написанного на одной машине, на другой, в случае если на этих машинах под один и тот же тип данных выделяется разный объем памяти. Чтобы его предотвратить, лучше строить алгоритм так, чтобы не приближаться к ограничениям по памяти, но можно также использовать вспомогательные типы данных, для которых объем выделяемой памяти зафиксирован независимо от архитектуры процессора.
Объявление - заявление о функции или переменной которая будет использоваться без указания полной информации о ней(только имя и тип). Может происходить неограниченное число раз. Определение - по сути объявление, содержащее полную информацию. Может происходить только один раз. Операция присваивания - процесс заполнения переменной информацией, которая в ней будет храниться. Может происходить неограниченное количество раз. Инициализация - первое присваивание совершенное над переменной с момента ее объявления. Соответственно, происходит оно только один раз.  
Дело в том что процессор может вытаскивать из памяти только блоки определенной величины и при отсутствии выравнивания данных по этим блокам ему или придется выгружать дополнительную информацию, либо передавать управление ОС, что понижает производительность.
